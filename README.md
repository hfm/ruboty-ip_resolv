# Ruboty::IpResolv

Resolve IP Address to Hostname

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty'
gem 'ruboty-ip_resolv'
```

And then execute:

    $ bundle

## Usage

See `ruboty help` or the following:

- [lib/ruboty/handlers/ip_resolv.rb](lib/ruboty/handlers/ip_resolv.rb)
